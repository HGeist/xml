Twitter Query Test

Twitter bietet 3 verschiedene API-Endpoints an:

  1. Search API
  2. REST API
  3. Streaming API
Für unsere Zwecke sollte die Search API brauchbar sein. Mit jeder API sind allerdings nur die letzten Tweets erreichbar (letzte Woche maximal).

Die Search API bietet viele verschiedene Parameter, die alle HTML encoded werden müssen. Argumente werden durch Leerzeichen (%20) voneinander getrennt. Beispielparameter sind : #hastag und @mention, also z.B. ist eine Query, die alle letzten Tweets über Berlin folgendermaßen aufgebaut: http://search.twitter.com/search.json?callback=?&q=%40Berlin

In diesem Beispiel wird das Twitter Plugin für jQuery benutzt (ist ne ganze Ecke einfacher zu benutzen und sieht auch noch schicker aus)
